Solución de Expediente Electrónico Integrado

Arquitectura

Desde sus comienzos el SIU proveyó soluciones monolíticas orientadas a solucionar problemas de dominios específicos.

Estas soluciones se instalaban directamente sobre el SO asumiendo que todas las dependencias se encontraban provistas por el entorno:

  • Servidor Web
  • Versión específica de PHP
  • Módulos de PHP
  • Otras dependencias de SO

Integración

Hace ya varios años se viene trabajando en dirección hacia la integración de todos los modulos del SIU logrando buenos resultados.

APIs Rest

Todos nuestros módulos cuentan con una API Rest de gestión.

Araí

Se han desarrollado módulos específicos para solventar necesidades básicas

  • Araí-Usuarios (IdP e IdM)
  • Huarpe (Portal unificado)
  • Araí-Documentos (Archivos y workflow de firma)

😊

Hay más sistemas, que si bien son más chicos, los requerimientos de instalación son los mismos que para los módulos tradicionales.

Crece la complejidad de despliegue y condiciona la utilización de nuevas tecnologías.
